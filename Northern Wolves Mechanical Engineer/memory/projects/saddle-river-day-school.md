## Key M-002 Specifications

### Piping
- **HW Piping ≤2":** Copper Type 'L' soldered
- **HW Piping ≥2.5":** Schedule 40 steel welded
- **Condensate Drain:** Copper Type 'L' soldered wrought copper (NOT PVC)
- **Refrigerant Piping:** 500 PSI rating, forged brass, brazed connections, per IMC Ch.11 §1107/1108, ASHRAE 15

### Pipe Supports
- 1-1/4" & smaller: 6' O.C.
- 1-1/2" & larger: 10' O.C.
- Copper: 8' O.C.

### Hangers
- Clevis, pipe roll, and pipe clamp types
- FNW or equal

### Pipe Sleeves
- Steel Type E, Grade A, Schedule 40, galvanized, plain ends

### Ductwork
- Galvanized sheet metal per SMACNA
- Joints ≥12": Ductmate system
- Seal Class A
- Flex duct: Thermaflex Type M-KE

### Insulation Schedule
| Application | Temp Range | Conductivity |
|------------|-----------|-------------|
| Hot Water | 141–200°F | 0.25–0.29 |
| Condensate Drain | 40–60°F | 0.21–0.27 |
| Refrigerant | 40–60°F | 0.21–0.27 |

### Seismic
- Site Class C, SRC III, Ip=1.0
- Ss=0.365G, S1=0.071G
- Wind speed: 123 mph

## Key Findings / Decisions
- PVC is NOT allowed for condensate drain piping — M-002 Note 12.A.C. specifies Copper Type 'L'
- Section 232113 (Hydronic Piping) does not exist in the DIV 23 spec book — all piping specs are on M-002 drawing
- Submittals must include project-specific callouts (type, size, standards, drawing references), not generic descriptions
