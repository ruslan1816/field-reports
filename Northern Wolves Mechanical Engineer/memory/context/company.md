# Company Context — Northern Wolves Air Conditioning

## About

| Service | Scope |

## Submittal Standards
- Cover page with project info, spec section, and detailed material tables
- Must highlight project-specific materials: type, size, color, standards
- Reference exact drawing notes (e.g., "per M-002 Note 12.A.C.")
- Attach manufacturer cut sheets from Standard Submittals Library
- Track in Excel submittal log
