# Glossary

Workplace shorthand, acronyms, and internal language for Northern Wolves AC.

## HVAC Acronyms
| Term | Meaning | Context |
|------|---------|---------|
| RFI | Request for Information | Formal question to engineer/architect |
| RTU | Rooftop Unit | Packaged HVAC on roof |
| AHU | Air Handling Unit | Indoor air distribution unit |
| VRF | Variable Refrigerant Flow | Multi-zone refrigerant system |
| ERV | Energy Recovery Ventilator | Heat + moisture recovery |
| HRV | Heat Recovery Ventilator | Heat recovery only |
| DOAS | Dedicated Outdoor Air System | Fresh air handling |
| VAV | Variable Air Volume | Zoned airflow control |
| FPB | Fan Powered Box | Terminal unit with fan |
| WSHP | Water Source Heat Pump | Water loop heat pump |
| BMS/BAS | Building Management/Automation System | Controls system |
| MAU | Makeup Air Unit | Tempered replacement air |
| VFD | Variable Frequency Drive | Motor speed control |
| CFM | Cubic Feet per Minute | Airflow measurement |
| GPM | Gallons per Minute | Water flow measurement |
| MBH | Thousand BTU per Hour | Heating capacity |
| EWT/LWT | Entering/Leaving Water Temperature | Hydronic temps |
| SRC | Seismic Risk Category | Building code classification |

## Specifications & Standards
| Term | Meaning | Context |
|------|---------|---------|
| DIV 23 | CSI Division 23 | HVAC Specifications section |
| M-002 | Mechanical Specifications drawing | Contains piping, ductwork, material requirements |
| SMACNA | Sheet Metal & Air Conditioning Contractors' National Assoc. | Ductwork standards |
| IMC | International Mechanical Code | Refrigerant piping code |
| ASHRAE 15 | Safety Standard for Refrigeration Systems | Refrigerant piping design |
| Seal Class A | Highest duct sealing classification | Per SMACNA |
| Ductmate | Flange connection system | For duct joints ≥12" |
